Welcome to Pyseams's documentation!
===================================
**Pyseams** is a Python package for `D-SEAMS <https://docs.dseams.info>`_ (Deferred Structural Elucidation Analysis for Molecular Simulations). It is a tool for the analysis of molecular dynamics trajectories which is specifically able to qualitatively classify ice structures in both strong-confinement and bulk systems.
